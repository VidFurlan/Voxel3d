# Voxel3d
3D voxel engine written in C++ with OpenGL.

##### Dependencies:
- OpenGL
- GLFW (has to be installed manually)
- GLAD
- GLM

## Roadmap
### Rednering optimizations
- Check voxels touching
- Split chunk into directional meshes
- Render only visible chunk meshes
